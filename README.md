# test-clock

A modern web-based clock application that displays real-time and date information.

## Features
- Real-time clock display (HH:MM:SS format)
- Current date and day of the week
- Responsive design for mobile devices
- Modern UI with gradient background and glassmorphism effects
- Automatic updates every second

## Technology Stack
- HTML5
- CSS3 (with modern features like backdrop-filter)
- Vanilla JavaScript (no external dependencies)

## File Structure
- `index.html` - Main HTML structure
- `styles.css` - CSS styling with responsive design
- `script.js` - JavaScript for time and date functionality

## Usage
Simply open `index.html` in a web browser to view the clock. The time will update automatically every second.

## Design Features
- Gradient background (purple to blue)
- Glassmorphism effect with backdrop blur
- Responsive layout that adapts to different screen sizes
- Clean and modern typography
- Subtle shadows and transparency effects
